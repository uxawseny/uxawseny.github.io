ApplicationContextпјҢBeanFactoryпјҢ IOC зҡ„е…ізі»

  • е…ізі»

    ApplicationContextжҳҜжҜ”BeanfactoryжӣҙеҠ е…Ҳиҝӣзҡ„IOCе®№еҷЁзҡ„дёҖз§Қе®һзҺ°

  • ApplicationContextзұ»еһӢе®№еҷЁжҸҗдҫӣзҡ„йӣҶдёӯе®һзҺ°

    FileSystemXmlApplicationContextпјҡд»Һж–Үд»¶дёӯеҠ иҪҪbeanе®ҡд№үд»ҘеҸҠзӣёе…іиө„жәҗзҡ„ApplicationContextе®һзҺ°гҖӮ

    ClassPathXmlApplicatonContextпјҡд»ҺClasspathеҠ иҪҪbeanе®ҡд№үд»ҘеҸҠзӣёе…іиө„жәҗзҡ„Applicationcontextзҡ„е®һзҺ°гҖӮ

    XmlWebApplicationContextпјҡз”ЁдәҺwebеә”з”ЁзЁӢеәҸзҡ„ApplicationContextзҡ„е®һзҺ°гҖӮ

  • ApplicationContextдёҺBeanFactoryзҡ„иҒ”зі»

    жҺҘеҸЈApplicationContextз»§жүҝдәҶListableBeanFactoryе’ҢHierarchicalBeanFactoryиҝҷдёӨдёӘжҺҘеҸЈпјҢ

    жҜҸдёҖдёӘBeanеҜ№еә”Springе®№еҷЁйҮҢзҡ„дёҖдёӘJavaе®һдҫӢпјҢе®ҡд№үBeanж—¶йҖҡеёёйңҖиҰҒжҢҮе®ҡдёӨдёӘеұһжҖ§гҖӮ

    @Component:@ComponentжҳҜжүҖжңүеҸ—Spring з®ЎзҗҶз»„д»¶зҡ„йҖҡз”ЁеҪўејҸпјҢ@ComponentжіЁи§ЈеҸҜд»Ҙж”ҫеңЁзұ»зҡ„еӨҙдёҠпјҢ@ComponentдёҚжҺЁиҚҗдҪҝз”ЁгҖӮ
@Controller:@ControllerеҜ№еә”иЎЁзҺ°еұӮзҡ„BeanпјҢд№ҹе°ұжҳҜActionгҖӮ
    @ControllerжіЁи§Јж ҮиҜҶUserActionд№ӢеҗҺпјҢе°ұиЎЁзӨәиҰҒжҠҠUserActionдәӨз»ҷSpringе®№еҷЁз®ЎзҗҶпјҢеңЁSpringе®№еҷЁдёӯдјҡеӯҳеңЁдёҖдёӘеҗҚеӯ—дёә"userAction"зҡ„actionгҖӮ
    иҝҷйҮҢзҡ„UserActionиҝҳдҪҝз”ЁдәҶ@ScopeжіЁи§ЈпјҢspring й»ҳи®Өscope жҳҜеҚ•дҫӢжЁЎејҸ(scope="singleton")пјҢиҝҷж ·еҸӘдјҡеҲӣе»әдёҖдёӘActionеҜ№иұЎпјҢжҜҸж¬Ўи®ҝй—®йғҪжҳҜеҗҢдёҖActionеҜ№иұЎпјҢж•°жҚ®дёҚе®үе…ЁпјҢstruts2 жҳҜиҰҒжұӮжҜҸж¬Ўж¬Ўи®ҝй—®йғҪеҜ№еә”дёҚеҗҢзҡ„ActionпјҢscope="prototype" еҸҜд»ҘдҝқиҜҒеҪ“жңүиҜ·жұӮзҡ„ж—¶еҖҷйғҪеҲӣе»әдёҖдёӘActionеҜ№иұЎгҖӮ
@Service:@ServiceеҜ№еә”зҡ„жҳҜдёҡеҠЎеұӮBean
@Repository:@RepositoryеҜ№еә”ж•°жҚ®и®ҝй—®еұӮBean
@AutowiredпјҡиҮӘеҠЁиЈ…й…ҚпјҢе…¶дҪңз”ЁжҳҜдёәдәҶж¶ҲйҷӨд»Јз ҒJavaд»Јз ҒйҮҢйқўзҡ„getter/setterдёҺbeanеұһжҖ§дёӯзҡ„property
    @Autowired й»ҳи®ӨжҢүзұ»еһӢиЈ…й…ҚпјҢеҰӮжһңжҲ‘д»¬жғідҪҝз”ЁжҢүеҗҚз§°иЈ…й…ҚпјҢеҸҜд»Ҙз»“еҗҲ@QualifierжіЁи§ЈдёҖиө·дҪҝз”ЁгҖӮеҰӮдёӢпјҡ
    @Autowired @Qualifier("personDaoBean") еӯҳеңЁеӨҡдёӘе®һдҫӢй…ҚеҗҲдҪҝз”Ё
@Resourceй»ҳи®ӨжҢүеҗҚз§°иЈ…й…ҚпјҢеҪ“жүҫдёҚеҲ°дёҺеҗҚз§°еҢ№й…Қзҡ„beanжүҚдјҡжҢүзұ»еһӢиЈ…й…ҚгҖӮ
@Scope("prototype")пјҡSpringй»ҳи®Өдә§з”ҹзҡ„beanжҳҜеҚ•дҫӢзҡ„пјҢ"prototype"иЎЁзӨәеҺҹеһӢеҚіжҜҸж¬ЎйғҪдјҡnewдёҖдёӘж–°зҡ„еҮәжқҘгҖӮ
@ConfigurationжҠҠдёҖдёӘзұ»дҪңдёәдёҖдёӘIoCе®№еҷЁпјҢе®ғзҡ„жҹҗдёӘж–№жі•еӨҙдёҠеҰӮжһңжіЁеҶҢдәҶ@BeanпјҢе°ұдјҡдҪңдёәиҝҷдёӘSpringе®№еҷЁдёӯзҡ„BeanгҖӮ
